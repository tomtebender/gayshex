# gayshex
**Utilities to bring more flow into your everyday terminal usage.**

**gayshex** is a package of various shell scripts to make your everyday easier. 
It's nothing special or extraordinary, gayshex is just a collection of scripts that run useful command combos.

## The Name
gayshex is a recursive acronym for "gayshex (gives you) shell extensions". 

Trust me, there is absolutely no different meaning. The name just came to me in a dream.
